# MNCJG 智能修复平台

椰壳生物炭/MnOx 复合材料的去除性能预测平台

## 功能特性

- 🎯 支持 Cd(II)、Pb(II)、萘等污染物的去除率预测
- ⚡ 实时计算，响应快速
- 📊 直观的结果展示
- 🎨 现代化界面设计
- 📱 完全响应式设计，支持各种设备

## 快速开始

### 本地开发

1. **克隆仓库**
   ```bash
   git clone https://github.com/YOUR_USERNAME/mncjg-platform.git
   cd mncjg-platform
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动开发服务器**
   ```bash
   npm start
   ```

4. **部署到 GitHub Pages**
   ```bash
   npm run deploy
   ```

## 模型参数

在 `src/App.js` 中修改 `MODEL_PARAMS` 对象的值。
